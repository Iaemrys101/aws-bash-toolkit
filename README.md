# AWS Bash Toolkit

A menu-driven Bash toolkit for managing AWS resources through the AWS CLI.

## Features

### Amazon S3

- Create S3 buckets
- List S3 buckets
- Upload files
- Download files
- Delete buckets

### Amazon EC2

- List EC2 instances
- Start EC2 instances
- Stop EC2 instances

## Requirements

- Linux or WSL
- Bash
- AWS CLI v2
- Configured AWS credentials
- View EC2 instance details

## Usage

Make the scripts executable:

```bash
chmod +x *.sh ec2/*.sh
